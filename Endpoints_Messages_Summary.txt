🔐 AUTHENTICATION
-----------------
POST /register              → 201 "User created successfully"
POST /login                 → 200 "User logged in" + session_token
POST /login (fout)          → 401 "Invalid credentials"
GET /logout                 → 200 Logout succesvol

👤 PROFILE  
----------
GET /profile                → 200 Gebruikersprofiel gegevens
PUT /profile                → 200 Profiel bijgewerkt
GET /profile (geen token)   → 401 "Unauthorized: Invalid or missing session token"
